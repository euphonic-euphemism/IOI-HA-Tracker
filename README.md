# IOI-HA Tracker (PDSA)

A React application for tracking International Outcome Inventory for Hearing Aids (IOI-HA) outcomes.

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
1. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally
Start the development server:
```bash
npm run dev
```

### Building
Build for production:
```bash
npm run build
```

## Project Structure
- `src/components`: UI components
  - `ui/`: Generic UI elements (Card, etc.)
- `src/hooks`: Custom hooks (`usePatients`)
- `src/data`: Static data and norms (`norms.js`)
